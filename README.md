# CRUD de Estudiantes en Consola

Proyecto en JavaScript que implementa un sistema CRUD para gestionar estudiantes en la consola.

## Características
Implementa operaciones CRUD (Crear, Leer, Actualizar, Eliminar).
- Utiliza ES Modules (import y export).
- Menú interactivo con readline-sync.
- Clase Estudiante con ID autoincremental.
- Clase GestorEstudiantes para manejar la lista de estudiantes.
- No usa map, filter, find, reduce, spread (...) ni destructuring.

## Instalación
1. Clona el repositorio:
   - git clone https://github.com/tu-usuario/crud-estudiantes-js.git

2. Instalar Dependencias:
   - npm install

3. Ejecutar el programa: 
   - node index.js
